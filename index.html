Metric definitions — takeoff▾
MetricUnitInterpretation
Jump Height (FT)cmFlight time method. Most sensitive fatigue indicator. >5% decline = watch, >10% = flag.
RSImodm/sJump height / contraction time. Primary neuromuscular readiness proxy. Population norms (Sole et al.): ≥0.40 optimal, 0.28–0.40 caution, <0.28 high risk. Athletes below 0.40 should be judged against own baseline — >8% drop = caution, >15% = high risk.
Concentric PF/BMN/kgPeak propulsive force relative to bodyweight. <10% of baseline = fatigue flag.
Eccentric PF/BMN/kgPeak braking force in downward phase. Drop = muscular fatigue or eccentric inhibition.
Peak Power/BMW/kgRate of energy output. Sensitive to accumulated fatigue.
Ecc DurationmsTime from onset to lowest point. Longer = slower unloading, reduced elastic energy.
Con DurationmsTime from lowest point to takeoff. Rising with falling JH = power loss.
FT:CT Ratio—Flight time / contraction time. Higher = more efficient neuromuscular output.
CM DepthcmNegative = downward displacement. Deeper = more energy stored but longer contraction time.
Asymmetry Index% L,R(L−R)/(L+R)×100. + = left dominant. >10% = flag, >15% = red.
VTOm/sVertical velocity at takeoff. JH(m) ≈ VTO²/(2×9.81).
Metric definitions — landing▾
MetricUnitInterpretation
PLF/BMN/kg>6×BW (58.8 N/kg) = elevated. >8×BW (78.5 N/kg) = high risk. Use LEI for full picture.
Landing Efficiency IndexN/kg per cmPLF/BM ÷ Jump Height. Normalizes landing force to height fallen. Lower = more efficient.
Landing RFDN/sRate of force development on landing. High RFD = stiff, rapid loading.
Landing ImpulseN·sForce × time. Higher = more compliant. Low impulse = rigid.
CoV %%>15% = inconsistent control. Needs 2+ trials.
Programming decisions
  • ▸RSImod <0.35: reduce intensity, prioritise sleep and recovery
  • ▸Ecc duration >20ms above baseline: add slow-tempo RDLs, eccentric-focused lifts
  • ▸FT:CT falling while JH holds: athlete maintaining output via longer contraction — not sustainable
  • ▸Asymmetry >15% sustained: screen for injury, add unilateral strength work
  • ▸PLF/BM >8×BW landing: pull landing-intensive work, add deceleration drills
  • ▸CoV >15%: inconsistent motor pattern — add deceleration drills before loading
